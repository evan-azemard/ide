Présentation de Visual Studio

Visual Studio est un ensemble complet d'outils de développement permettant de générer des applications web ASP.NET , des services web XML , des applications bureautiques et des applications mobiles.

Utilisez le débogueur Visual Studio pour détecter et corriger rapidement des bogues dans divers langages. Le débogueur Visual Studio pour Mac vous permet d’entrer dans votre code en définissant des points d’arrêt, des instructions Pas à pas principal, des fonctions Pas à pas détaillé et Pas à pas sortant, et d’inspecter l’état actuel de la pile de code à l’aide de visualisations détaillées.

Avec la puissance de Roslyn, Visual Studio pour Mac met IntelliSense à portée de clavier. IntelliSense complète automatiquement les descriptions des API lorsque vous les saisissez et utilise la saisie semi-automatique pour une plus grande rapidité et une plus grande précision lors de l’écriture du code.
